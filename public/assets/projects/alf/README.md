# Alf - Assets do Projeto

Esta pasta contém todos os assets do projeto Alf.

## Estrutura

```
alf/
├── images/
│   ├── cover/        # Imagem de capa do projeto
│   └── sections/     # Imagens para as secções do projeto
├── videos/           # Vídeos e animações
└── documents/        # PDFs e documentos
```

## Caminhos de referência

- Imagens: `/assets/projects/alf/images/sections/nome.jpg`
- Vídeos: `/assets/projects/alf/videos/nome.mp4`
- Documentos: `/assets/projects/alf/documents/nome.pdf`
