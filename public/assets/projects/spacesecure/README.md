# Spacesecure - Assets do Projeto

Esta pasta contém todos os assets do projeto Spacesecure.

## Estrutura

```
spacesecure/
├── images/
│   ├── cover/        # Imagem de capa do projeto
│   └── sections/     # Imagens para as secções do projeto
├── videos/           # Vídeos e animações
└── documents/        # PDFs e documentos
```

## Caminhos de referência

- Imagens: `/assets/projects/spacesecure/images/sections/nome.jpg`
- Vídeos: `/assets/projects/spacesecure/videos/nome.mp4`
- Documentos: `/assets/projects/spacesecure/documents/nome.pdf`
