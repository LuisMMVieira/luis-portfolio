# Images Directory

Esta pasta pode conter imagens gerais do site (não específicas de projetos).

## ⚠️ Nota Importante

**As imagens dos projetos foram movidas para `public/assets/projects/`**

Cada projeto agora tem a sua própria pasta completa em:

- `public/assets/projects/spaceship/images/`
- `public/assets/projects/alf/images/`
- `public/assets/projects/spacesecure/images/`

Ver `public/assets/README.md` para mais informações sobre a nova estrutura.

## Imagens gerais

Se precisares de imagens que não são específicas de um projeto (como logos, backgrounds gerais, etc.), podes colocá-las aqui.

## Formatos recomendados

- **Imagens**: `.jpg`, `.png`, `.webp`
- **Tamanhos**:
  - Desktop: até 1920px de largura
  - Mobile: até 800px de largura
- **Otimização**: Comprime as imagens antes de adicionar ao projeto
